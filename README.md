# hypertube

## Project Overview

## Setup

**Clone the Repository**
   ```bash
   git clone https://github.com/jguigli/hypertube.git
   cd hypertube
   ```

**Install the app**
   ```bash
   make build
   make
   ```
    

## Usage

- **Access hypertube**: Visit `http://localhost:3000` to access the hypertube site.
